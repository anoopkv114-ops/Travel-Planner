Travel Planner

Project Description:
Travel Planner is a web-based application that helps users explore travel destinations and plan trips efficiently. It provides essential information such as destination details, weather updates, location data, and interactive maps, making trip planning easier and more organized.

Problem Statement:
Planning a trip usually requires using multiple platforms to search destinations, check weather conditions, and view maps. This project solves that problem by providing a single platform where users can explore destinations, view important travel information, and plan their trips effectively.

Features:
- Destination search
- Weather and location information
- Interactive map integration
- Trip planning support

Technologies Used:
- HTML
- CSS
- JavaScript
- Maps API

Objective:
The objective of this project is to build a simple and user-friendly travel planning application using frontend web technologies and APIs while improving practical skills in web development.
